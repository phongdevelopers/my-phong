Tracker URL Help

You can use the following variables in the URL and they will be replaced with the dynamic value when the tracking url is rendered.

[OrderTotal]
Displays the order total.
[OrderSubTotal]
Displays the order total, less shipping and handling.
[OrderNumber]
Displays the order number.

Note that the variables are case sensitive.

AffiliateWiz Example:
https://www.yourdomain.com/affiliatewiz/aw.aspx?Task=AddSale&Order_Number=[OrderNumber]&Total_Amount=[OrderSubTotal]
